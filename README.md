+ FutureTask实现原理
    + 应用场景：在一个主线程中启动多个线程执行独立的任务 例如：存在多个文件，里面结构相同，去读之后需要汇总，那么此时就可以封装一个Task 通过runable或者callable接口实现多个子线程同时读取
    内容的逻辑，然后在主线程中进行合并，这样就会大大缩减单个文件读取一个个叠加。
